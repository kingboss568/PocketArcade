The first six games are free: Brick Blitz, Snake EVO, Stack Attack, Pong Duel, Memory Match, and Fruit Catch.

Premium games are gated by net.boss888.pocketarcade.unlockall. Remove ads product ID: net.boss888.pocketarcade.removeads.

This build does not include the GoogleMobileAds SDK; ad UI is protocol-gated and can stay hidden until a real provider is added. Arcade Coach currently demonstrates the required unavailable fallback and does not call an external AI service. Game Center leaderboards should use IDs named pocketarcade_<gameID>.
